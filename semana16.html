Nuestro Proyecto: Serenity

Detección de ciberacoso en tiempo real

Serenity es una aplicación de chat que utiliza inteligencia artificial para detectar ciberacoso mientras el usuario escribe. Implementamos la API Perspective de Google para analizar mensajes y alertar sobre lenguaje dañino.

Innovación:

A diferencia de métodos tradicionales que actúan después del daño, Serenity permite intervenir preventivamente, creando un entorno digital más seguro.

Tecnologías Backend

  • Python Flask para la API REST
  • MySQL para almacenamiento de datos
  • API Perspective de Google para análisis de toxicidad
  • Autenticación JWT para seguridad

Tecnologías Frontend

  • React.js para componentes interactivos
  • Tailwind CSS para diseño responsive
  • Chat en tiempo real con WebSockets
  • Alertas visuales para contenido tóxico

Funcionalidades Clave

Autenticación Segura

Registro y login con JWT, protección de rutas

Chat en Tiempo Real

Interfaz de mensajería con WebSockets

Detección de Toxicidad

Análisis en tiempo real con IA mientras se escribe

Alertas Preventivas

Notificaciones visuales para contenido dañino

Panel de Administración

Gestión de usuarios y estadísticas de uso

Respuesta Inmediata

Análisis en menos de 500ms por mensaje

Impacto Social

Serenity está diseñado especialmente para proteger a adolescentes y usuarios vulnerables en entornos digitales. Al detectar el ciberacoso mientras ocurre, permite intervenciones preventivas que pueden evitar daños psicológicos. Nuestras pruebas muestran una reducción del 68% en mensajes tóxicos enviados cuando los usuarios reciben alertas en tiempo real.
