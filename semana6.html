React - Componentes y Desarrollo con JavaScript Moderno

Exploramos React, una biblioteca declarativa y eficiente para construir interfaces de usuario basadas en componentes reutilizables.

Resumen de la Semana

React es una biblioteca de JavaScript creada por Facebook en 2011, diseñada para construir interfaces de usuario reutilizables. Uno de sus conceptos clave es el DOM virtual, que mejora el rendimiento al aplicar cambios en memoria antes de actualizar el DOM real.

Fundamentos de React

React permite crear interfaces eficientes basadas en componentes que gestionan su propio estado. Su enfoque modular promueve la reutilización de código y facilita el mantenimiento.

Mecanismos Internos

El DOM virtual permite a React minimizar manipulaciones del DOM real, lo cual mejora el rendimiento. React compara el nuevo estado con el anterior y actualiza solo los cambios.

Herramientas del Ecosistema

El ecosistema de React incluye herramientas como Create React App, Vite y Next.js. Estas simplifican la creación de proyectos y mejoran el flujo de trabajo del desarrollador.

Arquitectura de Proyectos

Una estructura típica de React incluye carpetas src/ y public/, y archivos como App.jsx y main.jsx que sirven de entrada principal.

Funciones Clave de React

useState()
const [count, setCount] = useState(0);

Permite declarar variables de estado.

useEffect()
useEffect(() => {...}, []);

Maneja efectos secundarios.

useContext()
const user = useContext(UserContext);

Comparte datos sin pasar props.

useReducer()
const [state, dispatch] = useReducer(...);

Para manejar estados complejos.
