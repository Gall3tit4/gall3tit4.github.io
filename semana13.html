Semana 13

Servidores Web, PHP y Laravel Framework

Esta semana exploramos el servidor web Apache, los fundamentos del lenguaje PHP y la creación de aplicaciones web modernas con el framework Laravel.

Resumen de la Semana

Configuramos un entorno de desarrollo completo con Apache como servidor web, PHP como lenguaje backend y Laravel como framework para desarrollo ágil de aplicaciones web.

1. Apache HTTP Server

El servidor web más popular para entregar contenido web:

Instalación con Apache Lounge

  1. Descargar binarios desde apachelounge.com
  2. Extraer ZIP en D:\Server\Apache24
  3. Configurar httpd.conf: 
    Listen 8080
ServerName localhost:8080
Define SRVROOT "D:\Server\Apache24"
  4. Ejecutar httpd.exe

Directivas principales

  • DocumentRoot: Directorio raíz del sitio
  • DirectoryIndex: Archivo índice (index.html, index.php)
  • ErrorLog: Ruta para logs de errores
  • LoadModule: Carga módulos adicionales

2. PHP: Lenguaje del lado del servidor

Instalación en Windows

  1. Descargar ZIP desde windows.php.net
  2. Extraer en D:\Server\PHP
  3. Configurar php.ini: 
    extension_dir = "ext"
extension=mysqli
extension=curl
  4. Agregar PHP al PATH del sistema
  5. Verificar con php -v

3. Laravel Framework

Framework PHP moderno para desarrollo web:

Instalación con Composer

# Crear nuevo proyecto
composer create-project laravel/laravel mi-proyecto

# Entrar al directorio
cd mi-proyecto

# Iniciar servidor
php artisan serve

Estructura de directorios

  • app/: Lógica de la aplicación
  • routes/: Definición de rutas
  • resources/: Vistas y assets
  • database/: Migraciones y seeds

Componentes principales

Eloquent ORM

Sistema de base de datos intuitivo

5. Flujo completo

  1. Request: Navegador solicita URL (Apache)
  2. Routing: Laravel determina la ruta correspondiente
  3. Controller: Ejecuta lógica de negocio
  4. Model: Interactúa con la base de datos
  5. View: Renderiza plantilla Blade con datos
  6. Response: Apache envía HTML al navegador

¿Qué Aprendí?

Comprendí la arquitectura de servidores web con Apache, los fundamentos del lenguaje PHP y cómo Laravel simplifica el desarrollo de aplicaciones web modernas con su conjunto de herramientas y convenciones. Aprendí a configurar un entorno completo y a implementar operaciones CRUD básicas.

¿Cómo Aprendí?

A través de la instalación práctica de cada componente, configurando manualmente Apache y PHP, y luego experimentando con Laravel para crear una aplicación funcional. La combinación de teoría y práctica me permitió entender cómo interactúan estas tecnologías en un proyecto real.
