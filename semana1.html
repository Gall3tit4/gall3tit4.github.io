Semana 1

Desarrollo Frontend y Backend de Aplicaciones Web

Un recorrido por los conceptos básicos de HTML y CSS para construir aplicaciones web modernas.

Resumen de la Semana

En esta semana, la temática principal fue el desarrollo teórico frontend y backend de aplicaciones web. Se abordaron conceptos fundamentales sobre las tecnologías clave que forman la base del desarrollo web moderno.

Desarrollo Web

Tecnologías Frontend

Se profundizó en las tres tecnologías fundamentales para el desarrollo frontend:

  • HTML: Lenguaje de marcado para estructurar el contenido web.
  • CSS: Hojas de estilo para el diseño y presentación visual.
  • JavaScript: Lenguaje de programación para agregar interactividad.

Frameworks y Librerías

Se exploraron diversas herramientas que facilitan el desarrollo:

  • Librerías como jQuery para simplificar el manejo del DOM.
  • Frameworks JavaScript MVC como Angular para aplicaciones más complejas.
  • Herramientas como Emmet para acelerar el desarrollo.

Desarrollo Backend

En la parte del servidor, se cubrieron aspectos esenciales:

  • Gestión de la lógica de negocio y procesamiento de datos.
  • Interacción con bases de datos para almacenar y recuperar información.
  • Configuración y manejo de servidores web.

Herramientas y Estándares

Se introdujeron conceptos importantes para el desarrollo profesional:

  • Estándares internacionales (W3C) para garantizar compatibilidad.
  • Control de versiones con Git y GitHub para trabajo colaborativo.
  • Entornos de desarrollo como Visual Studio Code.

Ejercicios de Laboratorio - Resultados

  1. Reconocer Herramientas de desarrollo integrado Visual Studio Code.
  2. Uso de GitHub Codespace.
  3. Uso de la Terminal de Visual Studio Code. Ejemplo práctico de git: git init, git add, git commit, git checkout, git push. Ramas y merge en GitHub.

¿Qué Aprendí?

Aprendí las bases esenciales del desarrollo web: cómo estructurar contenido con HTML, aplicar estilos visuales con CSS y agregar interactividad usando JavaScript. También conocí herramientas como Visual Studio Code y GitHub que mejoran la productividad y el trabajo colaborativo.

¿Cómo Aprendí?

Aprendí mediante sesiones teóricas y ejercicios prácticos en laboratorio. Usé entornos reales como GitHub Codespaces, realicé commits en la terminal, exploré el código fuente de páginas web y experimenté con herramientas modernas, todo en un enfoque activo y participativo.
