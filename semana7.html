Hooks en React

Uso de Hooks en componentes funcionales para crear interfaces dinámicas y reutilizables con React.

Resumen de la Semana

Los Hooks en React son funciones que permiten a los componentes funcionales manejar estado, efectos secundarios y más, sin usar clases. Fueron introducidos en la versión 16.8 de React y transformaron la forma de desarrollar componentes.

Introducción a los Hooks

Permiten utilizar características de React como estado y ciclo de vida en componentes funcionales. Simplifican el código, mejoran la reutilización de lógica y promueven una estructura más limpia.

Hooks más comunes

useState()
const [count, setCount] = useState(0);

Maneja el estado local dentro del componente.

useEffect()
useEffect(() => {...}, []);

Gestiona efectos secundarios como llamadas a API.

useContext()
const value = useContext(Contexto);

Permite acceder a valores globales del contexto.

useRef()
const inputRef = useRef(null);

Referencias a elementos o valores persistentes.

useReducer()
const [state, dispatch] = useReducer(...);

Estado complejo con lógica tipo Redux.

Combinación de Hooks

Los hooks pueden combinarse dentro de un mismo componente para construir lógica sofisticada y reutilizable, o agruparse en Custom Hooks.

Ejemplo de Custom Hook

function useContador(inicial) {
  const [valor, setValor] = useState(inicial);
  const incrementar = () => setValor(v => v + 1);
  return [valor, incrementar];
}
