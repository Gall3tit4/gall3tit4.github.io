Visual Studio Code y Emmet

Exploramos las herramientas esenciales del editor VSCode, la paleta de comandos, terminal y Emmet para un desarrollo ágil.

Resumen de la Semana

Esta semana nos enfocamos en dominar Visual Studio Code, el editor más usado para desarrollo web, desde su instalación hasta el uso de extensiones y atajos con Emmet.

Visual Studio Code

  • Instalación: Fácil para Windows, macOS y Linux.
  • Interfaz: Exploración del explorador, barra lateral, terminal.
  • Paleta de comandos: Ctrl + Shift + P para acceso rápido.
  • Terminal integrado: Ejecutar comandos directamente en VSCode.

Productividad con Emmet

  • Abreviaciones HTML: div.container>ul>li.item*5
  • CSS rápido: m10+p20 para márgenes y paddings.
  • Wrap with Abbreviation: Envolver bloques de código con estructuras HTML.
  • Navegación rápida: Ctrl + Alt + → entre elementos.
