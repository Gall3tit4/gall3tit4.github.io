Semana 15

Desarrollo Web de Sistemas Inteligentes

Implementación de sistemas inteligentes con Python, Flask y Django, integrando modelos de IA en aplicaciones web.

Resumen de la Semana

Esta semana exploramos el desarrollo de sistemas inteligentes capaces de percibir su entorno, procesar información, razonar y ejecutar acciones de forma autónoma, integrando estas capacidades en aplicaciones web modernas.

Arquitectura de Sistemas Inteligentes

Diagrama de Componentes

Los sistemas inteligentes modernos siguen una arquitectura modular que permite la integración de capacidades cognitivas avanzadas:

Módulo de Percepción

  • Interfaces con sensores físicos (IoT, cámaras)
  • Receptores de datos digitales (APIs, streams)
  • Preprocesamiento y normalización de datos
  • Conversión a representaciones internas
  • Técnicas avanzadas: Computer Vision, NLP
Ejemplo: Drones con sensores multiespectrales para agricultura de precisión

Módulo de Conocimiento

  • Bases de conocimiento estructuradas
  • Ontologías y grafos semánticos
  • Modelos de representación del dominio
  • Bases de datos vectoriales para embeddings
  • Memoria a corto y largo plazo
Ejemplo: Grafo de conocimiento médico con relaciones entre síntomas, enfermedades y tratamientos

Módulo de Razonamiento

  • Motores de reglas (Rule Engines)
  • Redes bayesianas para incertidumbre
  • Lógica difusa para variables cualitativas
  • Algoritmos de planificación (A*, STRIPS)
  • Sistemas expertos basados en conocimiento
Ejemplo: Sistema de diagnóstico médico que combina reglas clínicas con probabilidades bayesianas

Módulo de Aprendizaje

  • Redes neuronales (CNN, RNN, Transformers)
  • Algoritmos de aprendizaje supervisado y no supervisado
  • Refuerzo aprendizaje (Reinforcement Learning)
  • Transfer learning y fine-tuning
  • AutoML para automatización del proceso
Ejemplo: Modelo de visión por computadora que mejora su precisión con nuevos datos de entrenamiento

Flujo de Procesamiento

  1. Adquisición: Los sensores capturan datos del entorno (imágenes, sonido, señales)
  2. Preprocesamiento: Limpieza, normalización y transformación de los datos brutos
  3. Representación: Conversión a formatos adecuados para análisis (vectores, grafos)
  4. Análisis: Aplicación de modelos de ML y algoritmos de razonamiento
  5. Decisión: Generación de conclusiones o planes de acción
  6. Ejecución: Implementación de acciones a través de actuadores o interfaces
  7. Retroalimentación: Aprendizaje a partir de los resultados obtenidos

Patrones Arquitectónicos Comunes

Arquitectura basada en agentes

Sistemas multi-agente donde cada componente especializado colabora para resolver problemas complejos mediante comunicación y negociación.

Pipeline de procesamiento

Flujo secuencial de datos a través de módulos especializados (ej: visión → NLP → razonamiento).

Arquitectura cognitiva

Modela procesos humanos como memoria, atención y aprendizaje (ej: SOAR, ACT-R).

Sistema híbrido

Combina enfoques simbólicos (reglas) y subsimbólicos (redes neuronales) para mayor flexibilidad.

Ejercicios de Laboratorio - Resultados

  1. Algoritmos de toma de decisiones IF-THEN: Implementamos un sistema de diagnóstico médico basado en reglas. 
    def diagnosticar(sintomas):
    if "fiebre" in sintomas and "tos" in sintomas:
        return "Posible gripe"
    elif "dolor cabeza" in sintomas and "mareos" in sintomas:
        return "Posible migraña"
    else:
        return "Consulta médica recomendada"
  2. Control difuso de climatización: Sistema inteligente que ajusta la potencia del aire acondicionado usando lógica difusa.
  3. Interfaz web para LLM (Llama 3): Implementamos una aplicación Flask que interactúa con el modelo de lenguaje.

¿Qué Aprendí?

Aprendí a diseñar arquitecturas para sistemas inteligentes, implementando módulos de percepción, razonamiento y acción. Dominé técnicas como redes neuronales, algoritmos de decisión basados en reglas y lógica difusa. Además, integré estos sistemas en aplicaciones web usando Flask y Django, incluyendo modelos avanzados como Llama 3. Comprendí cómo los diferentes componentes (percepción, conocimiento, razonamiento y aprendizaje) interactúan para crear sistemas autónomos capaces de resolver problemas complejos.

¿Cómo Aprendí?

A través de ejercicios prácticos implementando sistemas reales: desde algoritmos de diagnóstico médico hasta controles difusos para climatización. Experimenté con la integración de LLMs en aplicaciones web, configurando entornos con transformers y diseñando interfaces de usuario interactivas. El aprendizaje fue hands-on, resolviendo problemas concretos de implementación mientras analizaba casos de estudio de sistemas inteligentes en diversos dominios.
