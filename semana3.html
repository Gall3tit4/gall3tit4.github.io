Semana 3

Emmet y Atajos en Visual Studio Code

Esta semana se aprendió a utilizar Emmet y los atajos de teclado más importantes para mejorar la eficiencia al programar en HTML/CSS.

Resumen de la Semana

Se aprendió a utilizar el plugin Emmet para escribir HTML y CSS más rápido, dominar atajos de teclado en Visual Studio Code y generar estructuras complejas usando abreviaciones.

Emmet: Abreviaciones Eficientes

  • >: Elementos hijos
  • +: Elementos hermanos
  • ^: Subir nivel
  • *: Repeticiones
  • {}: Contenido de texto
  • []: Atributos
  • $: Numeración

Ejemplo: ul.nav>li.item$*5>a[href="#"]{Opción $}

Atajos de Teclado Esenciales en VSCode

Generales

  • Ctrl + P: Buscar archivo
  • Ctrl + Shift + P: Paleta de comandos
  • Ctrl + , : Configuración

Edición Básica

  • Ctrl + X: Cortar línea
  • Ctrl + D: Duplicar línea
  • Alt + ↑ / ↓: Mover línea

Multicursor

  • Ctrl + Alt + ↑ / ↓: Múltiples cursores
  • Ctrl + Shift + L: Seleccionar ocurrencias
  • Alt + Click: Cursor manual

Ejercicios de Práctica

  • nav.nav>ul>li*4>a[href=page$@-1.html]{Link$@-1}
  • table>thead>tr>th{Id}+th{Nombre}+th{Apellido}^tbody>tr*3>td*3
  • div.card>img[src="img.jpg"]+div.card-body>h5{Producto}+p{Descripción}+a.btn{Comprar}

¿Qué Aprendí?

Aprendí a optimizar el tiempo de desarrollo web mediante el uso de abreviaciones con Emmet y dominé varios atajos de teclado en Visual Studio Code. Esto facilita la escritura de código limpio, rápido y eficiente.

¿Cómo Aprendí?

Mediante teoría, ejemplos prácticos, ejercicios guiados y el uso continuo de herramientas reales como VSCode y Emmet. La repetición y la práctica constante fueron claves para interiorizar lo aprendido.
